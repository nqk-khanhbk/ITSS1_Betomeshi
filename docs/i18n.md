# Hướng dẫn sử dụng i18n

Hệ thống đa ngôn ngữ được xây dựng dựa trên `react-i18next`.

## Cấu trúc thư mục

```
frontend/
  public/
    locales/
      en/
        translation.json
      vi/
        translation.json
  src/
    i18n.ts          # Cấu hình i18n
    components/
      LanguageSwitcher.tsx # Component chuyển đổi ngôn ngữ
```

## Cách thêm ngôn ngữ mới

1. Tạo thư mục mới trong `public/locales/` (ví dụ: `fr`).
2. Tạo file `translation.json` trong thư mục đó.
3. Thêm ngôn ngữ vào `LanguageSwitcher.tsx`.

## Cách sử dụng trong Component

Sử dụng hook `useTranslation`:
- Khai báo const { t } = useTranslation();
- Tham số truyền cho t:
  - key_in_json: key trong file translation.json
  - (option): tham số động

```tsx
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { t } = useTranslation();

  return <h1>{t('key_in_json')}</h1>;
}
```

Với tham số động:

```json
// translation.json
{
  "welcome": "Welcome {{name}}"
}
```

```tsx
t('welcome', { name: 'John' })
```

Với component (ví dụ đậm/nghiêng):
Dùng khi:
câu dịch có thẻ HTML mạnh/nhấn (strong/em),
hoặc có React component lồng trong text,
hoặc cần định dạng text theo ngôn ngữ.
Khi đó:
- file Json: 
```json
// translation.json
{
  "welcome": "Welcome <1>{{name}}</1>"
}
sẽ được tự động markup theo JSX ở trong strong.

s
```
```tsx
import { Trans } from 'react-i18next';


<Trans i18nKey="key_with_tags">
  Hello <strong>World</strong>
</Trans>
```
