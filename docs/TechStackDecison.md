# Tech Stack Decision

###Công nghệ
| Thành phần | Công nghệ | Lý do sử dụng |
|-------------|----------------|--------------------------------------------------------------------------------|
| Frontend | React | Các thành viên đều đã có kinh nghiệm. Hiệu năng tốt, dễ mở rộng, phù hợp bài toán nhiều request. |
| Backend | Express | Dễ triển khai, mạnh cho web thiên về thông tin (giới thiệu món ăn, nhà hàng). |
| Database | PostgreSQL | Các thành viên đã quen dùng; dữ liệu phù hợp mô hình quan hệ dạng bảng. |
