# べとめし

**Betomeshi** là một website giới thiệu các món ăn Việt Nam dành cho người dùng Nhật Bản, xây dựng chủ yếu với ngôn ngữ TypeScript,JavaScript.

## Mục đích dự án

- Lan tỏa văn hóa ẩm thực Việt Nam đến cộng đồng người Nhật.
- Cung cấp thông tin chi tiết về các món ăn, cách chế biến, nguyên liệu và đề xuất địa điểm thưởng thức tại Nhật Bản.
- Giao diện thân thiện, dễ dùng, có hỗ trợ ngôn ngữ Nhật.

## Công nghệ sử dụng

- **TypeScript** (ngôn ngữ chủ đạo)
- **JavaScript**
- **CSS**
- **PLpgSQL**
- **HTML**

## Hướng dẫn cài đặt và sử dụng

1. **Clone dự án:**
   ```bash
   git clone https://github.com/Robin-hoo/betomeshi.git
   cd betomeshi
   ```

2. **Cài đặt các gói phụ thuộc:**
   ```bash
   npm install
   ```

3. **Chạy ứng dụng:**
   ```bash
   npm run dev
   ```

4. Truy cập website tại
